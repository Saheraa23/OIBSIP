Project Title

Online Examination System

Description

The Online Examination System is a Java Swing-based desktop application developed to conduct simple online exams with login authentication, multiple-choice questions, timer functionality, password update option, and automatic score calculation.

The application provides an interactive graphical user interface (GUI) for students/users to attempt exams efficiently.

Features
Login Authentication
Secure login system
Default credentials:
Username: admin
Password: 1234
Multiple Choice Questions
Displays one question at a time
Four options for each question
User can select one answer
Timer Functionality
Countdown timer of 60 seconds
Auto submission when time ends
Score Calculation
Automatically checks answers
Displays final score after submission
Password Update
User can update login password during the session
Logout Option
Allows user to safely exit the application
Technologies Used
Java
Java Swing
AWT Event Handling
Timer Class
Object-Oriented Programming
