Project Title

Online Reservation System

Description

The Online Reservation System is a Java Swing-based desktop application developed to manage railway ticket reservations and cancellations. The application provides a simple graphical user interface (GUI) for users to log in, reserve tickets, generate PNR numbers, and cancel reservations.

This project is useful for understanding:

Java Swing GUI
Event Handling
Object-Oriented Programming
Data Storage using HashMap
Basic Reservation System Logic
Features
Login Authentication
Secure login screen
Default credentials:
Username: admin
Password: 1234
Reservation Form

Users can:

Enter passenger details
Select train number
Auto-fetch train name
Choose class type
Enter journey details
Generate unique PNR number
Cancellation Form

Users can:

Search reservation using PNR number
View reservation details
Cancel ticket after confirmation
Technologies Used
Java
Java Swing
AWT
Event Handling
Collections Framework (HashMap)
