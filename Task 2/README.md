Project Title

Number Guessing Game

Description

The Number Guessing Game is a Java Swing-based GUI application where the player attempts to guess a randomly generated number within a limited number of tries.

The game provides hints such as:

Too High
Too Low
Correct Guess

Players earn points for successful guesses and can restart the game using the reset button.

Features
Random Number Generation
Generates a random number automatically
Number range: 1 to 50
Guess Validation
Checks whether the guessed number is:
Correct
Too High
Too Low
Limited Attempts
Player gets 5 tries
Game ends when tries reach zero
Score System
Correct guess awards:
20 points
Reset Option
Starts a new game
Resets tries
Generates a new random number
Technologies Used
Java
Java Swing
AWT Event Handling
Random Class
Object-Oriented Programming
